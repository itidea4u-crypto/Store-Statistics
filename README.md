# Store Statistics

Dashboard แสดงสถิติร้านค้า — สำหรับทีมเซลและผู้จัดการ

## สถานะปัจจุบัน

- [x] หน้า dashboard พื้นฐาน (mock data)
- [x] การ์ดสรุป: Total Leads, Open Bills, Conversion Rate, Total Sales
- [x] ตารางรายชื่อเซล พร้อม conversion bar
- [x] รองรับมือถือ (responsive)
- [ ] เชื่อม Google Sheet API
- [ ] กราฟรายวัน / รายสัปดาห์
- [ ] ระบบ LINE Notify
